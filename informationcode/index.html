Pridėti atskirą css failą

Pridėkite apačioje esantį kodą į savo svetainės head elementą.

< ! -- Aš naudojau "./style.css" nes mano css failas yra tame pačiame aplanke kaip ir html failas. -- >
< link rel = "stylesheet" href = "./style.css" >

Jūsų svetainės kodas turėtu atrodyti maždaug taip.

< ! DOCTYPE html >
< html lang = "en" >
< head >
< meta charset = "UTF-8" >
< meta name = "viewport" content "width=device-width, initial-scale=1.0" >
< title > Document < / title >
< ! -- Aš naudojau "./style.css" nes mano css failas yra tame pačiame aplanke kaip ir html failas. -- >
< link rel = "stylesheet" href = "./style.css" >
< / head >
